WU Open Data Tool

Экспорт, проверка и аккуратное объединение данных открытых соревнований.

Рабочие ключи: open_athletes, open_freestyle_results, open_base_results

Правило безопасности

MacBook с базой и управлением соревнованиями считаем золотым источником. Этот инструмент не меняет данные сам по себе. Запись в localStorage выполняется только кнопкой «Применить merged» после ввода подтверждения.

Быстро через облако

На Mac с фристайлом отправь данные в облако. На основном Mac загрузи их как Source и запусти dry-run merge.

1. Экспорт текущего компьютера

Открой эту страницу на том же адресе GitHub Pages, где открывался протокол, и скачай снимок данных.

2. Загрузка двух snapshots

Golden = MacBook. Source = другой компьютер с результатами фристайла.

3. Dry-run merge

Правила: участники и база берутся из Golden. Из Source добавляются только результаты фристайла. Матчинг идёт по id, затем по уникальному ФИО. Неизвестные участники не добавляются автоматически.

4. Применить merged на этом браузере

Перед применением автоматически скачается backup текущего браузера. После применения открой open.html и проверь протокол/призёров.
